Log Sentinel

Утилита для мониторинга лог-файлов в реальном времени. Автоматически обнаруживает паттерны ошибок, фильтрует шум и отправляет алерты при критических событиях в серверных логах.

Node.js TypeScript File System Regex DevOps

О проекте

Назначение

Log Sentinel разработан для разработчиков и DevOps-инженеров, которым необходимо отслеживать логи серверов в реальном времени. Утилита анализирует потоки логов, выявляет аномалии, паттерны ошибок и критические события, отправляя уведомления мгновенно.

Где используется

Применяется в production-среде для мониторинга веб-серверов, API-шлюзов, микросервисов. Интегрируется с CI/CD пайплайнами для отслеживания ошибок деплоя. Незаменим при отладке распределённых систем и анализе инцидентов.

Ключевые возможности

Анализ паттернов

Распознавание паттернов ошибок через регулярные выражения

Система алертов

Мгновенные уведомления при обнаружении критических событий

Производительность

Минимальное потребление ресурсов при потоковой обработке

Пример кода

// Log Pattern Matcher
const errorPattern = /ERROR|FATAL|CRITICAL/i;

fs.watchFile(logPath, (curr, prev) => {
  const newLines = readNewLines(logPath, prev.mtime);
  newLines.forEach(line => {
    if (errorPattern.test(line)) {
      sendAlert(line);
    }
  });
});
