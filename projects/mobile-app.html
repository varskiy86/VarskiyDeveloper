Mobile App

Кроссплатформенное мобильное приложение, разработанное на React Native с использованием TypeScript. Обеспечивает нативный пользовательский интерфейс на iOS и Android с единой кодовой базой.

React Native TypeScript React Navigation REST API Mobile

О проекте

Назначение

Mobile App — это кроссплатформенное приложение, которое позволяет пользователям взаимодействовать с сервисами портфолио прямо со смартфона. Включает навигацию между экранами, авторизацию, просмотр проектов и интерактивные элементы управления.

Где используется

Приложение может быть развёрнуто в App Store и Google Play. Подходит для демонстрации мобильных навыков разработки, использования в качестве клиентского приложения для API Task Orchestrator и других backend-сервисов портфолио.

Ключевые возможности

Кроссплатформенность

Единая кодовая база для iOS и Android платформ

Навигация

Стек навигации с табами и стековыми переходами

API интеграция

Взаимодействие с REST API для получения данных
