Portfolio Engine

Интерактивное портфолио в стиле системного терминала. Пользователи взаимодействуют с контентом через консольные команды, что создаёт уникальный опыт погружения в мир разработчика.

React TypeScript Framer Motion Vite Lucide Icons

О проекте

Назначение

Portfolio Engine — это креативный способ презентации навыков и проектов разработчика. Вместо стандартных страниц используется интерфейс командной строки, что выделяет портфолио среди конкурентов и демонстрирует техническую экспертизу.

Где используется

Идеально подходит для персонального сайта разработчика, демонстрации на собеседованиях, хакатонах и технических конференциях. Может быть развёрнут на GitHub Pages, Vercel или Netlify.

Ключевые возможности

Командный интерфейс

Управление через команды: help, about, skills, projects, contact

Анимации

Плавные переходы и эффекты с помощью Framer Motion

Адаптивность

Корректное отображение на всех устройствах и экранах

Пример кода

// Terminal Component
const handleCommand = (cmd: string) => {
  switch(cmd.toLowerCase()) {
    case 'help':
      return 'Available: about, skills, projects';
    case 'skills':
      return 'TypeScript, React, Node.js...';
    default:
      return 'Command not found';
  }
};
