Auth Guard

Модуль безопасности для современных веб-приложений. Реализует продвинутую схему аутентификации с использованием короткоживущих Access-токенов и долгоживущих Refresh-токенов на базе JWT.

Node.js JWT Express bcrypt Security

О проекте

Назначение

Auth Guard обеспечивает надёжную защиту веб-приложений от несанкционированного доступа. Система управляет сессиями пользователей, проверяет права доступа и защищает API-эндпоинты с помощью middleware-архитектуры.

Где используется

Интегрируется в любые веб-приложения, требующие аутентификации: SaaS-платформы, интернет-магазины, корпоративные порталы, микросервисные архитектуры. Может работать как самостоятельный микросервис.

Ключевые возможности

JWT токены

Access + Refresh токены с автоматическим обновлением

Хеширование

bcrypt для надёжного хранения паролей

Middleware

Защита роутов на уровне маршрутизатора

Пример кода

// JWT Middleware
const authGuard = (req, res, next) => {
  const token = req.headers.authorization?.split(' ')[1];
  if (!token) return res.status(401).json({ error: 'Unauthorized' });
  
  try {
    const decoded = jwt.verify(token, process.env.JWT_SECRET);
    req.user = decoded;
    next();
  } catch {
    res.status(403).json({ error: 'Invalid token' });
  }
};
